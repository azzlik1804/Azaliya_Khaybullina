 Random Task Generator

> Генератор случайных задач с графическим интерфейсом

 Автор
 Хайбуллина Азалия Ленаровна

 Описание
Приложение на Python с использованием `tkinter`, которое:
-  Генерирует случайные задачи из предопределённого списка
-  Поддерживает фильтрацию по категориям: `учёба`, `спорт`, `работа`
-  Сохраняет историю сгенерированных задач в `tasks.json`
-  Позволяет добавлять новые задачи с валидацией ввода
-  Интегрируется с Git для контроля версий

 Требования
- **Python**: версия 3.6 или выше
- **Библиотеки**: только стандартные (`tkinter`, `json`, `random`, `os`, `subprocess`)
- **Git**: установлен и добавлен в PATH (опционально, для кнопки "Git: Init & Commit")

> `tkinter` обычно предустановлен в Python. Если возникает ошибка импорта:
> - **Windows**: переустановите Python с опцией "tcl/tk"
> - **Linux**: `sudo apt install python3-tk`
> - **macOS**: `brew install python-tk`

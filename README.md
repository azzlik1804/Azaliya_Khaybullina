Random Task Generator

 Автор
Хайбуллина Азалия Ленаровна 
Описание
Графическое приложение на Python, которое генерирует случайные задачи из предопределённого списка. Поддерживает фильтрацию по категориям (учёба, спорт, работа), сохранение истории в JSON-файл и интеграцию с Git.

Возможности
- Генерация случайной задачи
- Фильтрация по категории
- Добавление новых задач
- Сохранение и загрузка истории задач
- Инициализация Git-репозитория

 Примеры использования
1. Запустите приложение: `python main.py`
2. Нажмите "Сгенерировать задачу" — появится случайная задача.
3. Выберите категорию в выпадающем списке для фильтрации.
4. Нажмите "Добавить новую задачу", чтобы добавить свою задачу с категорией.
5. История задач сохраняется в `tasks.json`.
6. Нажмите "Инициализировать Git", чтобы создать репозиторий и сделать первый коммит.

 Требования
- Python 3.6+
- Библиотеки: `tkinter`, `random`, `json`, `os`, `subprocess` (все встроенные)
